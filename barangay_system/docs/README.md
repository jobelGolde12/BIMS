# Barangay Information and Management System (BIMS)

## Project Overview
BIMS is a comprehensive desktop application designed to streamline the management of resident information, household data, barangay officials, and the issuance of various certificates.

## Features
- User Authentication (Login System)
- Resident Management (CRUD, Search, Photo Upload)
- Household Management
- Barangay Officials Tracking
- Automated Certificate Generation (PDF)
- System Reports (CSV Export)
- Database Backup & Restore
- Activity Logging

## Folder Structure
- `barangay_system/`: Main application package
  - `database/`: SQLite database and schema
  - `ui/`: Tkinter GUI modules
  - `services/`: Business logic and services
  - `exports/`: Generated PDFs and CSVs
  - `docs/`: Documentation files
  - `charts/`: System diagrams
